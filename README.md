# 🌱 AGRO NEXUS 2050

## 🚀 O Futuro Brota da Terra

AGRO NEXUS 2050 é um site futurista desenvolvido com HTML, CSS e JavaScript que apresenta uma visão moderna do agronegócio sustentável. O projeto utiliza uma identidade visual baseada em verde-lima, preto e branco, transmitindo inovação, tecnologia e compromisso com o meio ambiente.

---

## 🌎 Slogan

**"Produzir mais. Preservar melhor. Conectar tecnologia, sustentabilidade e inovação para alimentar o futuro do planeta."**

---

## ✨ Funcionalidades

- Interface futurista com efeito neon
- Fundo animado com gradientes dinâmicos
- Navegação suave entre seções
- Layout totalmente responsivo
- Cards informativos sobre tecnologia agrícola
- Estatísticas animadas
- Botões interativos
- Efeitos visuais modernos
- Design focado em sustentabilidade e inovação
